Factorial Formula [Microsoft Python Interview Question]



In case you forgot the factorial formula, 

n!=n∗(n−1)∗(n−2)∗.....2∗1.

For example, 

5!=5∗4∗3∗2∗1=120 so we'd return 120.

Assume is 
n
n is a non-negative integer.
